# タスク管理システム / Task Management System

チームで共有できるタスク管理システムです。GitHub Issues と Projects を使って、誰でも簡単にタスクを作成・管理できます。

---

## 使い方 / How to Use

### タスクを作る（Issue を作成する）

1. 上部の **[Issues]** タブをクリック
2. **[New Issue]** ボタンをクリック
3. テンプレートを選んで記入
4. **[Submit new issue]** で作成完了

### ボードでタスクを確認する（Projects）

1. 上部の **[Projects]** タブをクリック
2. **「タスクボード」** を開く
3. カンバン形式でタスクの進捗を確認・更新できます

---

## ラベル一覧 / Labels

| ラベル | 意味 |
|--------|------|
| `priority: high` | 優先度：高 |
| `priority: medium` | 優先度：中 |
| `priority: low` | 優先度：低 |
| `type: bug` | バグ修正 |
| `type: feature` | 新機能追加 |
| `type: task` | 一般タスク |
| `type: question` | 質問・相談 |
| `status: in progress` | 作業中 |
| `status: blocked` | ブロック中 |
| `status: review` | レビュー待ち |

---

## タスクのステータス / Task Status (カンバンボード)

```
[Backlog] → [Todo] → [In Progress] → [Review] → [Done]
```

- **Backlog**: 将来やること
- **Todo**: 次にやること
- **In Progress**: 現在作業中
- **Review**: レビュー・確認中
- **Done**: 完了

---

## コラボレーター（共同作業者）を追加する方法

1. このリポジトリの **[Settings]** タブを開く
2. 左メニューから **[Collaborators]** を選択
3. **[Add people]** でユーザー名またはメールを入力
4. 相手が承認するとアクセスできるようになります

---

## Issue テンプレートについて

- **タスク**: 通常の作業タスク用
- **バグ報告**: 不具合の報告用
- **機能提案**: 新機能のリクエスト用
